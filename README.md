# Programación lineal
## Trabajo Final - Algebra lineal
### Objetivo
Pida el ingreso de una función lineal de la forma 𝑓(𝑥, 𝑦) = 𝑎𝑥 + 𝑏𝑦 y de n inecuaciones lineales (n[3, 8]) que formen un polígono D (acotado o no). El programa debe determinar el valor mínimo y máximo de la función sobre el conjunto D (en caso existan).
